# AI Creator - Design Specifications from Paper Mockups

## Layout Structure

### Top Navigation Bar
- Title area (left)
- Navigation tabs in center:
  - NN-AI (Neural Network AI)
  - NN-Storage (Neural Network Storage)
  - AI-H (AI Hybrid)
  - AI-B (AI Batch)
  - AI-Lab
  - AI-Prompt
- Logo area (right)

### Main Content Area - Two Column Layout

#### Left Column (AI-Usage)
- Shows AI model statistics
- ELA writer: 1M - 11%
- 16M64%
- Performance metrics display

#### Center Column (AI-Stage)
- ELA-writer: 4GB
- AI-42: (some metric)
- Large graph area showing performance trends
- Multiple colored lines representing different metrics

#### Right Sidebar
- More data
- More defined graphs
- Tree data that happened
- Page selected
- Neural Network storage
- AI-Creator creations and its creations

## Key Features to Implement

1. **Tabbed Navigation** - Multiple AI types/stages
2. **Performance Graphs** - Line charts showing metrics over time
3. **Data Display Panels** - Show AI usage statistics
4. **Sidebar** - Additional information and controls
5. **Color Coding** - Different colors for different AI types/metrics

## Color Scheme (from QIX Logo)
- Primary: Neon Green (#00DD00)
- Background: Dark (#141419)
- Accent: Light Gray (#C8C8C8)
- Graph lines: Multiple colors (blue, red, green, purple)

## Responsive Design
- Must handle fullscreen properly
- Content should resize with window
- Graphs should scale appropriately
